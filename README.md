# SchoolManagementSystem

Application de gestion scolaire développée avec :

## Backend
- Django
- Django REST Framework
- SQLite
- JWT Authentication

## Frontend
- React
- Vite
- Axios

## Fonctionnalités
- Gestion des étudiants
- Gestion des classes
- Gestion des enseignants
- Gestion des matières
